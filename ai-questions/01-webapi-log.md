## 質問と回答の記録

### Q1

**質問：**
このアプリで使われている Codable とは何ですか？

**AIの回答の要点：**
Codable は JSON データと Swift の構造体を相互に変換するための仕組みである。API から取得したデータを簡単に Song 構造体へ変換できる。

**自分の理解：**
Tunes API から受け取った JSON を Song や SearchResponse に変換するために Codable を使っていることが分かった。API 通信ではよく使われる重要な機能だと思った。

### Q2

**質問：**
searchMusic() 関数はどのような処理をしていますか？
**AIの回答の要点：**
入力された検索文字列を URL エンコードし、iTunes Search API にアクセスする。その後、取得した JSON データを解析して songs 配列に保存している。
**自分の理解：**
検索ボタンを押すと searchMusic() が実行され、API から曲情報を取得して画面に表示していることが理解できた。URLSession と JSONDecoder の役割も分かった。
### Q3

**質問：**
AsyncImage の役割は何ですか？
**AIの回答の要点：**
AsyncImage はインターネット上の画像を非同期で読み込み、表示するための SwiftUI コンポーネントである。画像が読み込まれるまでプレースホルダーを表示できる。
**自分の理解：**
このアプリではアルバムジャケット画像を表示するために AsyncImage が使われている。画像の読み込みを自動で行ってくれるので便利だと思った。
